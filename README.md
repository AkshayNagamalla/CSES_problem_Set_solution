To successfully submit Java files on the CSES website, you need to:

- Name the Java file as `Main.java`.
- Ensure the main class in your code is also named `Main`.

For example:

```java
// Main.java
public class Main {
    public static void main(String[] args) {
        // Solution code here
    }
}
